Certificate File

When initializing the server, you will at some point be required to generate a certificate file, which is used to encrypt and decrypt the SSL communication channel used by rhn_register and up2date. This certificate file must also be copied to all the client that will use the new Current server for retrieving and installing package updates. We will assume you place that file in /usr/share/rhn with the name CURRENT-CA-CERT for the remainder of this document.
